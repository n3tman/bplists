BSaber Playlist Formatter

This page checks and auto-formats your Beat Saber playlists. Use it before uploading them to BSaber.

Maximum accepted file size is 1.2 MB. If your .bplist is bigger, please resize the cover image.
For base64 conversion you can use Code Beautify website (copy full "img src" value with "data:image").
Small cover image (max 40 KB) is needed for the master list of playlists.
To fix errors because of removed maps, check if they've been reuploaded and update hashes in .bplist

Open your playlist file and press Submit button below.
